Marked.js Venn Diagram Plugin Demo

This demo shows how to use the Venn Diagram Plugin with Marked.js to create interactive Venn diagrams in your markdown content.

Note: The plugin creates a three-circle Venn diagram with content in the intersection areas. You can customize width, height, and fontSize.

Basic Example

```venn
A∩B | B∩C | A∩C | A∩B∩C
```

Named Parameters

```venn width=800 height=500 fontSize=16
Data | Logic | Design | All Three
```

HTML Content Example

```venn width=600 height=400 fontSize=14
A∩B | B∩C | A∩C | A∩B∩C
```

Mixed Font Sizes Example

```venn width=700 height=500 fontSize=12

Small Text
 | 
Medium Text
 | 
Large Text
 | 
Extra Large

```
