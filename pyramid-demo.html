Marked.js Pyramid Diagram Plugin Demo

This demo shows how to use the Pyramid Diagram Plugin with Marked.js to create interactive pyramid diagrams in your markdown content.

Note: The plugin creates a 4-layer pyramid with content in each layer. You can customize width, height, and fontSize.

Basic Example

```pyramid
Base | Middle | Upper | Top
```

Named Parameters

```pyramid width=500 height=400 fontSize=14
Foundation | Strategy | Tactics | Goal
```

HTML Content Example

```pyramid width=500 height=400 fontSize=14
Data Collection | Data Processing | Data Analysis | Insights
```

Mixed Font Sizes Example

```pyramid width=500 height=400 fontSize=12

Many Customers
 | 
Some Prospects
 | 
Few Leads
 | 
One Sale

```
